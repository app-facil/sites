Como Adicionar Seus Próprios Sites

1

Crie a Estrutura de Pastas

No seu repositório GitHub Pages, crie uma pasta sites/ e dentro dela, pastas para cada site.

2

Adicione os Arquivos

Coloque os arquivos de cada site em sua respectiva pasta. Cada site deve ter um index.html na raiz de sua pasta.

3

Atualize Esta Página

Edite este arquivo index.html para adicionar um card com link para seu novo site.

4

Acesse Seus Sites

Seus sites estarão disponíveis em https://seuusuario.github.io/sites/nomedosite
